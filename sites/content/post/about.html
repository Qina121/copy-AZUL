关于Azul Systems

我们是提升版Java。

Azul Systems 服务于需要以下方面的公司：为 Web 用户提供基于服务器的 Java 应用程序（旅行、在线销售、游戏、SaaS）；需要达到具体延迟目标水平的实时业务系统（广告网络、资本市场、通信）；需要保障关键应用能得到及时的维护和安全更新，或者需要为嵌入式或 IoT 用例提供基于 Java 的系统。Azul是世界上唯一专注于Java和Java虚拟机（JVM）的公司。Azul成立于2002年，是一家私营公司，总部位于加州桑尼维尔，并在全球设有办事处和支持中心。我们在2005年推出Vega，一款硬件支持的Java设备。在那之后，我们的业务重心转移到基于软件的平台和产品。我们于2010年发布 Azul Zing
